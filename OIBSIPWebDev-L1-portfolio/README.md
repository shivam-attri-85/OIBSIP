# Task 2 - Personal Portfolio

Responsive personal portfolio built with HTML5 and CSS3 based on the provided resume.

## Features

- Hero section with name, role title, and avatar placeholder
- About Me section with a short resume-based summary
- Skills grid showing technical tools and languages
- Projects section with placeholder GitHub links
- Contact section with name, email, LinkedIn, and GitHub
- Smooth scroll navigation
- Fully responsive layout

## Files

- `ShivamKumar_Task2.html` - main portfolio page
- `ShivamKumar_Task2.css` - styling and responsive layout

## How To Run

1. Open `ShivamKumar_Task2.html` in a browser.
2. Use the navigation links to jump between sections.
3. Resize the browser to check the responsive behaviour.
